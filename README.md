# 未来音乐坊 · 各校教学风采共享平台

深圳市妇女儿童发展基金会「未来音乐坊」公益项目 — 覆盖全国24所乡村和少数民族学校的音乐教育风采展示平台。

## 页面

- **index.html** — 风采展示页（学校卡片、照片灯箱、视频播放、分类筛选）
- **teacher.html** — 老师上传页（手机端友好，照片/视频上传）
- **admin.html** — 管理后台（数据管理、月度汇总、CSV导出、附件打包下载）

## 技术栈

- 前端：原生 HTML + CSS + JavaScript
- 后端：Node.js + Express
- 数据库：腾讯云 CloudBase 文档型数据库
- 存储：腾讯云 CloudBase 云存储
